Go to **Settings > Extensions** and click the `INSTALL` button on the `AWS Lambda Function` Extension. A Drawer similar to the picture below will pop up.

<img alt='Enable AWS Extension' width='auto' height='auto' src='https://cdn.apito.io/media/apito_documentation/RYZ8GISWTN_Apito_Console___AWS_Credentials.png' />

You will need an `Access Key ID` and `Secret access key` in-order to enable this Extension. Please refer to [this doc](/docs/extensions/logic/configuring-aws) to create
a new key if you do not have any. Now click `Save` button below to Enable the Extension. It will how a Success Message if properly enabled, and your aws key is valid
otherwise you will get an error message.
